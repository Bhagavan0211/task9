## Task 9: Credit Card Fraud Detection using Random Forest

### Objective
Detect fraudulent credit card transactions using machine learning while handling class imbalance.

### Models Used
- Logistic Regression (Baseline)
- Random Forest Classifier

### Key Concepts
- Imbalanced data handling
- Stratified train-test split
- Precision, Recall, F1-score
- Ensemble learning

### Results
Random Forest performed better than Logistic Regression in detecting fraud cases.

### Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib
